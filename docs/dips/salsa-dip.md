# Salsa dip

## Zutaten

| Menge | Zutat                          |
| ----- | ------------------------------ |
| 400 g | gehackte Tomaten aus der Dose  |
| 1     | Zwiebel                        |
| 1     | Knoblauchzehe                  |
| etwas | Olivenöl                       |
| etwas | Salz, Pfeffer, Cayenne pfeffer |
| etwas | Chili                          |

## Zubereitung

1. Zwiebeln und Knoblauch mit Olivenöl anbraten
2. Tomaten dazu geben und mit Salz, Pfeffer, Cayenne Pfeffer und Chili würzen.
