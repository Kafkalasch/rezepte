# Cheese Cake

Ein Cheese Cake zum Einkochen und als Kühlkonserve im Kühlschrank gut haltbar (hat sogar nach 2 Monaten noch gut geschmeckt).

## Zutaten

Folgende Menge ergibt ca. 1,2 Liter

| Menge  | Zutat                |
| ------ | -------------------- |
| 500 g  | Frischkäse, vollfett |
| 200 ml | Sauerrahm            |
| 50g    | Zucker               |
| 1 Pkg. | Vanillezucker        |
| 1 TL   | Zitronenabrieb       |
| 1 EL   | Mehl ?               |
| 3      | Eier                 |
